# basics


## installation

``` 
tilman@t570:~/learn_py/zoo$ sudo apt install python3.8-venv
tilman@t570:~/learn_py/zoo$ python3 -m venv env
tilman@t570:~/learn_py/zoo$ source env/bin/activate
(env) tilman@t570:~/learn_py/zoo$
```


``` 
pip install pytest
pip install bottle
```


## git

``` 
git remote add origin git@github.com:entenschule/zoo.git
git branch -M main
git push -u origin main
```

``` 
git status                         # check if changes are committed
git config --global user.name      # check user if in doubt
git push
```
